# To-Do List Application (CLI)

## Description
A simple console-based to-do list application built using Python.
Supports adding, viewing, and removing tasks with persistent file storage.

## Features
- Add tasks
- View tasks
- Remove tasks
- File-based storage (tasks.txt)
- Persistent data

## Technologies
- Python
- File Handling
- Lists
- CLI Interface

## How to Run
```bash
python todo.py
